## 📖 项目简介

**搜建筑导航** 是一个专为建筑行业从业者打造的高效规范查询与管理工具。

它结合了在线规范的便捷性与本地管理的私密性，并通过 **IndexedDB** 技术实现了**本地 PDF 的永久存储**，解决了传统网页刷新丢失文件的问题。我们还内置了强大的**本地文件管理器**，让您可以像操作电脑文件夹一样管理浏览器中的规范文件。

![项目预览](homepage_preview.png)

## ✨ 核心亮点 (v2.0 Updates)

### 1. 📂 强大的本地 PDF 系统
- **持久化存储**：集成了浏览器原生数据库 (IndexedDB)，**刷新页面、重启浏览器，您上传的 PDF 永远都在**。
- **本地文件管理器**：
  - 在卡片编辑中点击“浏览本地数据库”，即可打开可视化管理面板。
  - 查看所有已保存 PDF 的文件名、大小、上传时间。
  - 支持**一键下载**备份到本地硬盘，或清理不需要的文件。
- **目录导入自动保存**：通过“规范目录”批量上传的 PDF 也会自动存入数据库，确保数据无缝衔接。

### 2. 🎨 极致的交互体验
- **拖拽排序**：所见即所得，随意拖拽卡片调整顺序，或在分类间移动。
- **可视化编辑**：全新设计的编辑弹窗，支持修改名称、链接，以及直观的 PDF 文件管理。
- **一站式搜索**：顶部搜索栏支持实时过滤，毫秒级定位目标规范。

### 3. 📚 丰富的规范目录库
- **内置目录**：预置了上百条常用建筑规范数据（国标/行标/地标）。
- **批量导入**：
  - 支持 **Excel 导入**，快速建立企业级规范库。
  - 支持 **PDF 批量上传**，自动添加到分类并持久化保存。

### 4. 💾 数据安全与隐私
- **纯本地运行**：所有数据（配置、PDF文件）均存储在您的浏览器本地，**绝不上传到任何云端服务器**。
- **JSON 存档**：支持将您的所有配置导出为 `.json` 存档文件，方便备份或迁移到其他设备。

## 🛠️ 技术栈
- **Frontend**: React 18, Tailwind CSS
- **Storage**: IndexedDB (Dexie-like logic), localStorage
- **Icons**: FontAwesome 6
- **Build**: Single HTML file (No build step required for simple usage)

## 🚀 快速开始

本项目为纯静态单页应用，开箱即用。

1.  **下载/克隆项目**
    ```bash
    git clone https://github.com/qiushanyueyue/soujianzhu.git
    ```
2.  **运行**
    - 直接双击打开 `index.html` 即可完整体验所有功能。
    - 推荐使用 VS Code Live Server 获得最佳体验。
