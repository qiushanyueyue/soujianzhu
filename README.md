## 📖 项目简介

**搜建筑导航** 是一个专为建筑行业从业者打造的高效规范查询与管理工具。它不仅收录了丰富的建筑工程、设计、施工及验收规范，还创新性地支持**本地 PDF 文件的上传与管理**，助您打造个人专属的规范知识库。

![项目预览](homepage_preview.png)

## ✨ 核心功能

### 1. 🔍 极速检索
- 实时搜索过滤，毫秒级响应。
- 支持关键词模糊匹配，快速定位目标规范。

### 2. 📂 智能分类管理
- **拖拽排序**：所见即所得，通过拖拽轻松调整规范顺序与分类布局。
- **自定义分类**：支持创建、重命名、折叠分类，满足个性化整理需求。
- **状态记忆**：自动保存分类的折叠/展开状态，刷新页面不丢失。

### 3. 📄 本地 PDF 集成 (亮点)
- 打破在线浏览限制，支持**上传本地 PDF 文件**。
- 本地规范与在线规范统一展示，点击即可在浏览器中预览。
- 数据仅保存在浏览器本地，安全无忧。

### 4. 💾 数据安全与迁移
- **一键存档**：将当前的所有分类、链接及本地关联配置导出为 JSON 文件。
- **随时恢复**：支持导入存档文件，在不同设备间同步您的个性化配置。

## 🚀 快速开始

本项目是一个纯静态的单页应用 (Single Page Application)，无需复杂的后端部署。

1.  **克隆项目**
    ```bash
    git clone https://github.com/qiushanyueyue/soujianzhu.git
    cd soujianzhu
    ```

2.  **本地运行**
    - 方式一：直接用浏览器打开 `index.html` 文件。
    - 方式二（推荐）：使用 VS Code 的 Live Server 插件或 Python 启动本地服务。
      ```bash
      # Python 3
      python -m http.server 8000
      ```
      访问 `http://localhost:8000` 即可。
